Aqui estão algumas informações que podem lhe ajudar nessa decisão?
Qual a segurança nos relatórios? O Embedded possui camadas de segurança para que seus relatórios sejam visualizados apenas no seu portal. O usuário acessa com qual licença? O acesso do usuário é controlado pelo seu portal ou aplicativo. Apenas uma linceça é necessária para diversos usuários usarem o Power BI Embedded Consigo segmentar os dados dos relatórios por usuário? Sim! Através do RLS (Row Security Level) é possível atribuir perfis de acesso para cada usuário e segmentar o acesso aos dados dos reports. Posso utilizar o portal web que eu já tenho? Isso vai depender do seu portal. A Microsoft distribui bibliotecas de integração para tecnologias modernas como .Net, Node.js, Python, React.js, Vue.js, Angular. Se sua aplicação se enquadra em uma dessas, há uma grande chance de ser possível utilizar o Embedded. Caso seja uma outra tecnologia, nossos consultores estão preparados para fazer uma análise de viabilidade.
